# AI Job Salary Predictor 💼

A full-featured Streamlit web app that predicts salaries for AI/ML jobs, classifies them as high/low, and provides market insights with beautiful visualizations.

## 🚀 Features

- **Single Prediction**  
  Enter job details (title, experience, skills, location, etc.) → get:
  - Predicted salary
  - High/Low salary category
  - Market position (e.g., "Above Average / Top Tier")
  - 5-year salary growth projection with interactive chart

- **Bulk Prediction**  
  Upload a CSV with multiple job listings → get batch predictions + downloadable results

- **Market Insights Dashboard**  
  Explore salary statistics:
  - Average, median, and highest salaries
  - Salary distribution histogram
  - Average salary by industry (bar chart)

## 🛠 Tech Stack

- **Frontend**: Streamlit
- **Models**: LightGBM + scikit-learn pipelines
- **Visualization**: Plotly
- **Deployment**: Streamlit Community Cloud (free)
